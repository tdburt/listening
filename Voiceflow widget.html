Step 2: Host the HTML File
Upload this HTML file to a web hosting service:
GitHub Pages (free and easy)
Netlify (free)
Vercel (free)
Your own web server
Make sure the URL is HTTPS (required by TalentLMS).
Step 3: Embed in TalentLMS
Log in to TalentLMS as an Instructor or Administrator
Open your course
Click Add Standard Content iFrame
Paste your hosted HTML page URL (e.g., https://yourusername.github.io/voiceflow-
chat.html)
Name the unit (e.g., "Course Assistant")
Click Publish unit
Option 2: Use Pop-up Mode (If Embedding Fails)
If the iframe embedding doesn't work due to security restrictions:
Open the unit options (three-dot menu)
Choose Show as Pop-up
This will open your Voiceflow chat in a new window instead
Option 3: Enable Frame Loading (Advanced)
If you have Administrator access, you can enable frame loading:
Go to Account & Settings Security
Enable "Allow the application to be loaded in a frame"
This may help with embedding restrictions
Passing Student Data to Voiceflow (Optional)
If you want to personalize the experience with student information, you can modify the
HTML to pass variables:
window.voiceflow.chat.load({
verify: { projectID: 'YOUR_PROJECT_ID' },
url: 'https://general-runtime.voiceflow.com',
versionID: 'production',
render: {
mode: 'embedded',
target: document.getElementById('voiceflow-chat-frame')
},
autostart: true,
launch: {
event: {
type: 'launch',
payload: {
student_name: 'John Doe',
course_name: 'Introduction to AI'
}
}
}
});
